# Dashboard Analytics BRIDA JATIM

This project is a data analytics and visualization system for BRIDA (Regional Research and Innovation Agency) of East Java. It provides an interactive dashboard to analyze regional innovation data, including innovation maturity levels, categories, institutions, and geographical distribution.

## AI Workflow Architecture using n8n

## Tech Stack
1. **Frontend:** 
- React.js / Vite
- Tailwind CSS

2. **Backend:**
- FastAPI

3. **Database:**
- Supabase

